# ClaudeManager

`~/.claude` を読んで、**スキル・サブエージェント・プロジェクトメモリ・設定・稼働統計**を1画面で可視化し、GUIで作成・編集・削除できるローカルWebアプリ。

## コンセプト

- **最小限の依存関係**: ビルド不要のVanilla JS + Express（依存は実質1つ）で構成され、どこでも軽量に動作します。
- **Claude Codeを中心としたフォールバック設計**: `claude` CLIを軸にしつつ、GrokやAgyなどの他のエージェントCLIにも対応済みです（Codex等は必要に応じてユーザー自身で拡張可能）。
- **どこからでもフルアクセス**: TailscaleなどのVPN環境と組み合わせることで、スマホや外出先のブラウザからでも自宅マシンのエージェント環境にフルアクセスし、指示管理が可能です。
- **あなた専用のマネージャー**: 本アプリをベースにカスタマイズすることで、幅広いエージェント管理を実現できます。まだ成長途中のため、OSSとして公開し、自由な改変・活用を推奨しています。

## 起動

```bash
# 自分のマシンだけで使う（既定・安全）
npx claude-manager --open

# LAN / Tailscale 経由でスマホ等から使う（トークン自動発行）
npx claude-manager --host :: --open
```

ローカル開発時:
```bash
npm install
npm start            # http://localhost:4317（localhostのみ）
```

オプション: `--port <n>` `--host <addr>` `--token <str>` `--open`

## 機能

- **Skills** — ローカル(`~/.claude/skills`)＋プラグインのSKILL.mdを一覧・検索。新規作成・編集・削除。
- **Subagents** — ユーザー定義＋プラグインのエージェント定義を一覧・作成・編集。
- **Projects** — 各プロジェクトのCLAUDE.md・memory（MEMORY.md＋個別メモリ）を表示・編集、メモリ新規作成。
- **Config** — 全体指示 `~/.claude/CLAUDE.md` と `settings.json` を編集。
- **Assistant** — 相談しながらスキル・設定・プロジェクトを調整する**組み込みエージェント**（下記）。
- **Sessions** — 進行中・最近のClaude Codeセッションを可視化し、`--resume`で参加。
- **Overview** — 総セッション/メッセージ/ツール呼出、直近30日グラフ、設定概要、MCP認証待ち。

エディタはMarkdownプレビュー対応。保存・削除時は自動で `.bak` を退避。

## 組み込みアシスタント

「このスキルにこの注意を追記して」「このプロジェクトのCLAUDE.mdにデプロイ手順を足して」のような自然言語の指示で、`~/.claude` 配下やプロジェクトのファイルを編集します。

- 裏で **Claude Code（`claude` CLI）を headless 実行**。会話は継続（`--session-id`/`--resume`）。
- **相談モード（既定）**＝編集せず変更案を提示／**実行モード**＝実際に編集。提案に「この案で実行」ボタン。
- スキル・サブエージェント・プロジェクト指示など対象に応じて文脈を切り替え。
- モデルは **Haiku（軽量・既定）/ Sonnet / Opus** を選択可能。
- Claude Code が導入済み・ログイン済みであることが前提。

### Sessions

- `~/.claude/projects/*/<id>.jsonl`（セッション記録）を解析し、**進行中（直近更新）/最近**のセッションを一覧。
- cwd・直近のやりとり・ツール使用・イベント数を表示。クリックで詳細。
- 「このセッションに参加」＝ `claude --resume <id>` で履歴を引き継いでメッセージ投入（編集許可は任意）。
- 注: 起動中の対話画面へのライブ割り込みではなく、同一セッション履歴を共有する resume 方式。

## セキュリティ

- 既定では **localhost のみ** にバインド。
- `--host` で外部公開する場合は **トークン必須**（未指定なら自動発行し、URLに付けて表示）。ループバックは無条件許可。
- ファイルの読み書きは **`~/.claude` 配下と `CLAUDE.md` のみ**に制限（任意ファイルの読み出し不可）。
- アシスタントはローカルのClaude Codeを実行するため、公開時はトークンを厳重に扱うこと。
